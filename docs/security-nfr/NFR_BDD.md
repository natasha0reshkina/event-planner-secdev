Функция Производительность выборки событий
Сценарий Быстрый ответ на выборку по диапазону
Дано развернутый учебный стенд и нагрузка пятьдесят запросов в секунду
Когда запускается пятиминутный нагрузочный тест на конечную точку выборки событий с параметрами начало и конец
То девяносто пятый перцентиль времени ответа не выше двухсот миллисекунд

Функция Ограничение частых ошибок входа
Сценарий Блок после серии неверных попыток
Дано чистая база и адрес входа
Когда один адрес выполняет пять неверных входов в течение десяти минут
То следующая попытка в этот период получает отказ с пояснением про временную блокировку

Функция Доступ только владельцу
Сценарий Чужой ресурс не раскрывается
Дано два пользователя владелец и посторонний и созданное событие владельца
Когда посторонний запрашивает это событие
То сервис возвращает код 404 и в журнале есть запись с идентификатором запроса

Функция Валидация дат
Сценарий Запрет на прошлую дату
Дано попытка создать событие с датой раньше текущего дня
Когда отправляется создание такого события
То сервис возвращает код 422 с сообщением о неверной дате

Функция Дубликаты
Сценарий Запрет на повтор
Дано существующее событие с одинаковыми названием датой и местом
Когда отправляется создание второго такого же события
То сервис возвращает код 409 и не создаёт новую запись
